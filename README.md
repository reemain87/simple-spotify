# simple-spotify

simple-spotify is a collection of tutorials and examples for using the Spotify API.

## Features
- Access Spotify playlists and tracks
- Search for music by artist, album, or track name

## Technology Stack
- JavaScript
- Node.js
- Express

## Usage
1. Clone the repository
2. Install dependencies: `npm install`
3. Run the application: `node index.js`
